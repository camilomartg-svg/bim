# VSR_IFCA

Base para la variante Artis Urbano del visor `VSR_IFC`.

Proposito inicial:
- separar una segunda version del visor IFC
- aplicar identidad visual de Artis Urbano
- mantenerla independiente de `VSR_IFC`

Siguiente paso recomendado:
- clonar la estructura base de `VSR_IFC` dentro de `VSR_IFCA`
- reemplazar logos, colores y configuracion especifica de marca
